Quick Start Commands:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Start Backend:
   cd backend
   npm start

2. Start Frontend (new terminal):
   npm run dev

3. Create More Users:
   cd backend
   node seed/seedUsers.js

4. Access Application:
   Open browser to http://localhost:3000
   Login with any credentials above
